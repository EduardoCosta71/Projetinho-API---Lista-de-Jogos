Recentemente desenvolvi um projeto de API utilizando Flask em Python 

Nesse projeto, implementei um CRUD completo (Create, Read, Update e Delete), permitindo criar, listar, atualizar e remover informações de uma lista de jogos.

Com isso, consegui praticar na prática conceitos importantes como rotas, métodos HTTP (GET, POST, PUT e DELETE) e organização de uma aplicação back-end.

Mesmo sendo um projeto simples, foi essencial para entender melhor como funciona a comunicação entre cliente e servidor.

Sigo evoluindo e buscando aprimorar meus conhecimentos em Python e desenvolvimento de APIs
